选择本地文件
aui.file.select(func, ones);

键 类型 默认 注释
func function * 委托函数, 参数为FILE类
ones bool true 选择多个文件，开始时参数为数组
读取本地文件
let file;
aui.file.select(function(e){
 file = e
});
let call = function(e){
 console.log(e)
};
aui.file.read(file, call, type, char);

键 类型 默认 注释
file FILE类 * 从读取文件返回的函数
call function * 委托函数
type string text 可选参数为text,byte,other。分别对应文本readAsText, 二进制readAsBinaryString, 其他readAsDataURL
char string UTF-8 文件编码, 仅text有效
读取本地文件

aui.file.readSelect(func, type, ex, msg);

键 类型 默认 注释
func function * 委托函数, 参数为文件内容
type string text 文件类型
ex string 文件后缀过滤('后缀1,后缀2'), 无视大小写
msg string 文件选择异常提示
