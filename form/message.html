aui.tips(msg, time, func)
键 类型 默认 注释
msg string * 提示信息

弹窗


aui.message(msg, param)
键 类型 默认 注释
msg string * 提示信息
param object 额外参数
param.name1 string 确定 按钮1文字
param.func1 function 按钮1事件
param.name2 string 返回 按钮2文字
param.func2 function 按钮2事件(默认不显示,需要显示请定义true或事件)
模仿移动端推送
 

var param = {
 onclick: function;
 msg: '信息';
};
var form = aui.push(param);
键 类型 默认 注释
msg string * 提示信息
icon string 图标链接
closeTime int 自动关闭时间(毫秒)
onclick function 点击事件
onclose function 关闭事件
clickClose bool true 点击后关闭
onload function 启动事件
