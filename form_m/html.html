图片集组件

<div id="img_list" class="list img-list" style="height: 33em;">
 <div class="img-list-item img-list-item-new"></div>
</div>

var param = {
 ex = 'png,jpg',
 onload = function,
};
var img_list = aui.html.img_list($('#img_list'), param)




附加参数
键 类型 默认 注释
ex string 'png,jpg' 图片过滤的后缀名
img_name bool false 是否允许编辑文件名
menu array 额外右键菜单, 与menu参数相同
onload function 启动事件

调用方法
函数 注释
img_list.count() 获取图片数量
img_list.clear() 清空图片
img_list.add() 选择添加新图片
img_list.get() 获取根据图片顺序的数据
{data: base64编码的图片, name: 图片名称, id: 编号, ex: 文件后缀, param: 额外数据}
