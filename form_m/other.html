背景轮播
let imgs = [ 'https://api.dujin.org/bing/1920.php' ]; aui.bgImage(imgs, time)

键 类型 默认 注释
imgs array * 图片URL数组
time int 30000 切换时间(毫秒)
定时器组件
函数 注释
aui.timer.start() 启动定时器
aui.timer.add(func, loop, next) 添加定时器
loop = 定时器间隔(秒)
next = 下次执行时间(默认为当前时间+间隔时间)
aui.timer.del(func) 删除定时器
aui.timer.clear() 清空定时器
aui.timer.stop() 停止定时器
COOKIE组件
函数 注释
aui.cookie.set(key, val, day) 设置cookie
key = 键
val = 值
day = 保存天数
aui.cookie.get(key) 获取cookie
aui.cookie.del(key) 删除cookie
数组组件
函数 注释
aui.array.remove(item, array) 数组删除第一个匹配元素
aui.array.isIn(item, array) 检测数组是否包含项
字符串组件
函数 注释
aui.string.trim(str) 清除首尾的特殊字符
aui.string.copy(str) 复制到剪贴板
aui.string.low(str) 字符串转小写
aui.string.upp(str) 字符串转大写
aui.string.isIn('a', str) 字符串是否包含指定字符
aui.string.pos('a', str) 指定字符在字符串中的位置
