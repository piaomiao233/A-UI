打开新窗口
var option = {
    formPath: 'function.html',
};
var form = new aui.form(option);

键 类型 默认 注释
form JQ对象 窗体对象，也可以直接定义
formPath url * 窗口html文件的路径,如果定义form则此项失效
clickitem JQ对象 触发的JQ对象,用于获取图标
close_func function 关闭时触发函数
parent JQ对象 父级窗口,暂无特殊用途
useMask bool false 使用遮罩层
maskBlack bool true 遮罩层黑色
maskClose bool false 点击遮罩层触发关闭
data 任意 用户自定义数据
icon url 在底部窗口栏中的图标链接
mini_icon url 在图标中额外添加一个小的图标
ones bool true 限制1窗口
onload func|array 启动后触发(参数为aui.form类),支持单func或func array
drag bool true 启动拖动
offset float 1 居中的Y轴比例, 0.8为80%窗口高度的一半
anime bool true 使用窗口动画
select bool true 美化html的select
checkbox bool true 美化html的checkbox
range bool true 美化html的range
mobile bool false 强制移动模式
menu bool true 初始化menu模块
tips bool true 初始化tips模块
窗体功能
函数 参数类型 注释
form.get(value) string 获取data中的数据(data必须是object)
form.find(value) string JQ的find,等同于form.form.find()
form.center(value) float 窗口居中，等同上方offset
form.show() 从最小化中显示窗口并置顶
form.mini() 窗口最小化
form.top() 窗口置顶
form.close() 关闭窗口
form.height() 获取窗口高度
form.width() 获取窗口宽度
form.offset() 获取窗口坐标
