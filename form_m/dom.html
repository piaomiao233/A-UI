窗口居中
aui.dom.center(dom, offset, back)
键 类型 默认 注释
dom JQ对象 窗体对象
offset float 1 垂直居中比例
back bool false 不居中,但是返回居中坐标
窗口拖动
aui.dom.drag(dom,dragDom,click,top)
键 类型 默认 注释
dom JQ对象 窗体对象
dragDom JQ对象 拖动触发区域，留空则全窗体触发拖动
click function 点击窗体时触发
top bool true 拖动时置顶
窗口移动到可视区域内
aui.dom.inwin(dom)
从左上角开始顺时针判定，如果超出边界则移动到窗口内
