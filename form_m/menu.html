普通菜单


var option = {
 list: [{name:名称, style=样式, func=事件}],
 onclose: function,
 event: mouseEvent,
};
var form = aui.menu(option);
键 类型 默认 注释
list array * 按键组{name:名称, style=样式, func=事件}
maskBlack bool false 遮罩层黑色
onclose function 关闭触发
offset float 高度参数,如果定义则居中对齐
event mouseEvent 鼠标事件, 如果定义则可以跟随鼠标
data 任意 玩家自定义数据
